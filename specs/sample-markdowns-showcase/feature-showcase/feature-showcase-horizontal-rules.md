# Horizontal Rules

This demonstrates horizontal rule (divider) functionality in markdown.

---

## Different Syntaxes

You can create horizontal rules using three or more hyphens, asterisks, or underscores:

### Using Hyphens
---

### Using Asterisks

***

### Using Underscores

___

---

## Multiple Separators

You can use more than three characters:

-----

*****

_____

---

## Usage

Horizontal rules are great for:
- Separating sections
- Creating visual breaks
- Organizing content
- Improving readability

---

## Features

- Styled with theme colors
- Consistent appearance across modes
- Subtle opacity for elegance
- Full-width rendering

---

## Syntax

```markdown
---
***
___

-----
*****
_____
```

All of these create the same visual horizontal rule.
