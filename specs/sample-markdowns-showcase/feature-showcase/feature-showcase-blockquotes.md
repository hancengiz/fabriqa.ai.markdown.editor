# Blockquotes

This demonstrates blockquote functionality in markdown.

---

## Simple Blockquotes

> This is a simple blockquote.
> It can span multiple lines.
>
> And include multiple paragraphs.

---

## Nested Blockquotes

> This is the first level of quoting.
>
>> This is nested blockquote inside another quote.
>>
>>> And this is triple-nested!
>>> You can nest as deep as you need.

---

## Blockquotes with Formatting

> **Bold text** in blockquotes
> *Italic text* in blockquotes
> `Code` in blockquotes

---

## Blockquotes with Lists

> Here's a list inside a blockquote:
> - Item 1
> - Item 2
> - Item 3

---

## Obsidian-Style Editing

In **Live Preview** mode:
- When cursor is **outside**: Styled blockquote appearance
- When cursor is **inside**: Raw markdown with `>` symbols visible
- This allows easy editing while maintaining visual clarity

---

## Syntax

```markdown
> Single line quote

> Multi-line
> quote

>> Nested quote
```
