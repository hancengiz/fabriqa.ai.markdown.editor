# Combined Features

This demonstrates how different markdown features can be combined together.

## Task List with Formatting

Combine task lists with various text formatting:

- [x] **Bold task** completed
- [ ] *Italic task* pending
- [x] Task with `inline code`
- [ ] Task with [link](https://example.com)
- [x] ~~Strikethrough completed task~~
- [ ] Task with ***bold italic*** text

> ```
>
> This is useful for documentation and examples.
