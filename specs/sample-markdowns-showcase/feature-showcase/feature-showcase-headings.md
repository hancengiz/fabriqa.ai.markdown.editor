# Headings

This demonstrates all heading levels supported in markdown.

---

# H1 Heading
## H2 Heading
### H3 Heading
#### H4 Heading
##### H5 Heading
###### H6 Heading

---

## Usage

Headings are created using hash (#) symbols:

- `# H1` - Largest heading
- `## H2` - Second level
- `### H3` - Third level
- `#### H4` - Fourth level
- `##### H5` - Fifth level
- `###### H6` - Smallest heading
