# Keyboard Shortcuts

This is a comprehensive guide to all keyboard shortcuts in fabriqa.ai markdown editor.

---

## Mode Switching

Switch between editing modes instantly:

- `Cmd+Shift+P` - **Live Preview Mode**
- `Cmd+Shift+S` - **Source Mode**
- `Cmd+Shift+R` - **Reading Mode**

---

## Text Formatting (Cmd+Alt+Key)

Apply formatting to selected text:

- `Cmd+Alt+B` - **Bold**
- `Cmd+Alt+I` - *Italic*
- `Cmd+Alt+C` - `Inline Code`
- `Cmd+Alt+X` - ~~Strikethrough~~
- `Cmd+Alt+K` - [Link]()
- `Cmd+Alt+E` - Code Block
- `Cmd+Alt+H` - Toggle Heading
- `Cmd+Alt+Q` - Blockquote
- `Cmd+Alt+T` - Task Checkbox
- `Cmd+Alt+7` - Numbered List
- `Cmd+Alt+8` - Bullet List

---

## Table Editing

### Navigation
- `Tab` - Move to next cell
- `Shift+Tab` - Move to previous cell

### Table Operations
- `Cmd+Shift+T` - Insert new table

### Row Operations
- `Cmd+Shift+↑` - Insert row above
- `Cmd+Shift+↓` - Insert row below
- `Cmd+Shift+D` - Delete current row

### Column Operations
- `Cmd+Shift+←` - Insert column left
- `Cmd+Shift+→` - Insert column right
- `Cmd+Shift+Backspace` - Delete current column

### Formatting
- `Cmd+Shift+F` - Format table (align columns)

---

## Search

Find and navigate through text:

- `Cmd+F` - Open find dialog
- `F3` - Find next occurrence
- `Shift+F3` - Find previous occurrence
- `Esc` - Clear find / close find dialog

---

## Standard Editing

Standard text editor shortcuts also work:

- `Cmd+C` - Copy
- `Cmd+X` - Cut
- `Cmd+V` - Paste
- `Cmd+Z` - Undo
- `Cmd+Shift+Z` - Redo
- `Cmd+A` - Select all
- `Cmd+S` - Save file

---

## Interactive Elements

### Checkboxes
- **Click** checkbox to toggle completion
- `Cmd+Alt+T` to create checkbox

### Links
- **Cmd+Click** on link to open
  - Markdown files open in editor
  - External URLs open in browser

### Mermaid Diagrams
- **Click inside** code block to edit
- **Click outside** to see diagram

---

## Tips for Efficiency

1. **Learn mode switching** - Fastest way to change context
2. **Use formatting shortcuts** - Much faster than typing syntax
3. **Tab in tables** - Quick navigation
4. **Cmd+F for search** - Find text anywhere
5. **Cmd+Click links** - Open without leaving editor

---

## Windows/Linux

Replace `Cmd` with `Ctrl` on Windows and Linux:
- `Ctrl+Shift+P` for Live Preview
- `Ctrl+Alt+B` for Bold
- etc.
