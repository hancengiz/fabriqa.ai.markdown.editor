# Text Formatting

This demonstrates various text formatting options in markdown.

---

## Bold Text

**Bold text** using double asterisks

__Bold text__ using double underscores

---

## Italic Text

*Italic text* using single asterisks

_Italic text_ using single underscores

---

## Bold and Italic

***Bold and italic*** using triple asterisks

___Bold and italic___ using triple underscores

**_Mixed bold and italic_**

---

## Strikethrough

~~Strikethrough text~~ using double tildes

---

## Combined Formatting

You can combine different formatting:

- **Bold** and *italic* together
- **Bold with ~~strikethrough~~**
- *Italic with ~~strikethrough~~*
- ***Everything ~~combined~~ together***
