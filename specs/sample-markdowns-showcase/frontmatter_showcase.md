# Frontmatter Showcase

The block above this header is the new **Frontmatter Widget**.

## Features Demonstrated
- **Strings**: Simple text overlapping.
- **Dates**: Standard ISO dates.
- **Booleans**: Rendered as checkboxes (read-only in Live Preview).
- **Arrays/Tags**: Rendered as colorful independent pills.
- **Long Text**: Handling of multiline descriptions.

Try clicking the "Properties" header to toggle visibility!
